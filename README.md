# Science Outline Writer

## Format

    I. Lesson Name
        A. Header Name
            1. Sentence
            2. Sentence
        B. Header Name
            1. Sentence
    II. Lesson Name
        A. Header Name
            1. Sentence

## How To Use

1. Type Lesson Name
2. Type Header Name
3. Type as many sentences as needed
4. To change to new header or lesson press ^C
5. Quit with ^C then Q
